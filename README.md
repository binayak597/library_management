# Library Management System (JAVA + Spring Boot)
This project allows librarians to manage book records.

## Features
✔ Add a book  
✔ View all books  
✔ Search book by ID or title  
✔ Update book details  
✔ Delete book  

## Setup
1. Clone the repository  
2. Run `mvn spring-boot:run`  
3. Use Postman to test APIs
